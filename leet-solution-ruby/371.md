# 371. Sum of Two Integers

Calculate the sum of two integers a and b, but you are **not allowed** to use the operator `+` and `-`.

**Example 1:**

```text
Input: a = 1, b = 2
Output: 3
```

**Example 2:**

```text
Input: a = -2, b = 3
Output: 1
```



```text
# @param {Integer} a
# @param {Integer} b
# @return {Integer}
def get_sum(a, b)
    [a,b].sum
end
```
